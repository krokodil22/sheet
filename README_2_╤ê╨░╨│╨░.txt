1. Кладете новые недельные выгрузки .xlsx в эту папку.
2. Запускаете "Запустить обновление.bat".

Результат:
- файл "Траектория_групп_обновляемая.xlsx" пересобирается заново;
- каждая группа остается в одной строке;
- каждая новая неделя добавляется вправо новым столбцом;
- если группы раньше не было, добавляется новая строка.

Важно:
- в папке должны лежать именно недельные выгрузки .xlsx;
- сам итоговый файл трогать не нужно, он обновляется автоматически;
- для запуска нужен установленный Python.
