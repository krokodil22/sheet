# Веб‑сервис объединения недельных таблиц

Этот репозиторий переведён из локального Python-скрипта в статический веб‑сервис, который можно опубликовать на **GitHub Pages**.

## Что делает сервис

- Загружает существующую общую таблицу (опционально).
- Принимает одну или несколько новых недельных `.xlsx` выгрузок.
- Объединяет данные так, чтобы каждая группа была в одной строке, а недели добавлялись отдельными столбцами.
- Формирует новый файл `.xlsx` для скачивания.

## Запуск на GitHub Pages

1. Загрузите файлы `index.html`, `app.js`, `styles.css` в репозиторий.
2. В GitHub: **Settings → Pages**.
3. В разделе **Build and deployment** выберите:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (или ваша ветка) и папка `/ (root)`
4. Сохраните. Через 1–2 минуты появится ссылка вида:
   `https://<username>.github.io/<repo>/`

## Локальная проверка

Можно просто открыть `index.html` в браузере.

## Технически

- Сервис полностью клиентский: данные не отправляются на сервер.
- Для чтения/записи Excel используется `SheetJS (xlsx)` через CDN.
